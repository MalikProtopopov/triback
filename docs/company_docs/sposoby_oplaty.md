# Способы оплаты

**Сайт:** [trichologia.ru](https://trichologia.ru/)

## Приём платежей через PayAnyWay

Онлайн-оплата на сайте обрабатывается платёжным сервисом **PayAnyWay** (НКО «Монета»). Данные банковских карт Ассоциации не получает: ввод реквизитов карты выполняется на защищённой стороне PayAnyWay.

---

## Доступные способы оплаты

Набор способов зависит от настроек платёжной страницы и может включать:

### Банковские карты

Оплата картами платёжных систем **МИР**, **Visa**, **MasterCard**, **UnionPay** (перечень на экране оплаты актуален на момент платежа).

### СБП (Система быстрых платежей)

Оплата через СБП без ввода данных карты: по QR-коду, кнопке или с привязанного счёта — в интерфейсе PayAnyWay.

### SberPay и иные pay-сервисы

При наличии в платёжной форме — оплата через приложение банка (например, SberPay) согласно условиям банка и PayAnyWay.

### Наличный расчёт

Используется только если для конкретной услуги или товара на сайте явно предусмотрена выдача при личной встрече / курьером / пункте выдачи. При получении проверьте комплектацию, при необходимости — документы и чек. В типовом сценарии (членские взносы, мероприятия, цифровые услуги) расчёт — **безналичный**, онлайн.

### Оплата по счёту (безналичный расчёт для юрлиц и ИП)

Для организаций и индивидуальных предпринимателей может быть доступна оплата по счёту, выставляемому Ассоциацией. Условия и реквизиты для перечисления согласовываются отдельно. Такой платёж может проходить **вне** интерфейса PayAnyWay (прямое зачисление на расчётный счёт по реквизитам из счёта). Срок поступления средств на счёт получателя — обычно в пределах нескольких рабочих дней после оплаты со стороны плательщика (зависит от банка).

## Безопасность платежей

- Страница оплаты работает по протоколу **HTTPS**.
- Данные карты **не передаются** интернет-магазину (Ассоциации) и третьим лицам — обработку выполняет PayAnyWay.
- Для карт поддерживается **3D Secure** (дополнительная аутентификация в банке).

Подробнее о защите данных — в документации PayAnyWay на официальном сайте сервиса.

## Возврат средств

Если оказана услуга или реализован товар **ненадлежащего качества**, возврат денежных средств возможен в порядке, установленном **ФЗ «О защите прав потребителей»**, на карту или кошелёк, с которых проводилась оплата (в рамках возможностей платёжной системы).

Конкретный порядок возврата по операциям через сайт Ассоциации — в разделе [Информация о возврате](https://trichologia.ru/) на сайте и по запросу на e-mail Ассоциации (см. страницу «Контактная информация»).
