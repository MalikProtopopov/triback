## Админка — чек-лист

- [ ] В списке врачей (`GET /admin/doctors`) показывать иконку «+ фото» при `has_photo_in_draft === true`
- [ ] В карточке врача при модерации черновика показывать блок «Новое фото», если `photo_url` в `changed_fields`
- [ ] URL превью: `${MEDIA_BASE_URL}/${pending_draft.changes.photo_url}` (см. [FRONTEND_MEDIA_URLS_HANDOFF.md](./FRONTEND_MEDIA_URLS_HANDOFF.md))

---

## Клиент — чек-лист

### Публичный сайт

- [ ] Страница врача: при 404 — «Врач временно недоступен» или 404
- [ ] Каталог: учитывать, что врачи без подписки не показываются

### Личный кабинет врача (ЛК)

- [ ] После `POST /profile/photo` показывать превью и статус «На модерации» (`pending_moderation: true`)
- [ ] В форме публичного профиля: при `pending_draft.changes.photo_url` показывать превью нового фото
