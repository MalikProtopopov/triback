# Голосование — Админ-панель

**Дата:** 2026-03-23  
**Кому:** Фронтенд-разработчик админ-панели  
**Тема:** Обновление подсчёта `total_eligible_voters` и автозакрытие сессий

---

## Что изменилось в бэкенде

1. **Подсчёт `total_eligible_voters`** — теперь учитываются только врачи с одобренным статусом **и** действующей подпиской (Subscription.status = active, ends_at > now или null).
2. **Автозакрытие сессий** — cron-задача каждый час переводит сессии с `ends_at < now` из `active` в `closed`. Ручное закрытие по-прежнему доступно.

---

## API

**Эндпоинт:** `GET /api/v1/admin/voting/{session_id}/results`

**Ответ без изменений структуры**, но семантика `total_eligible_voters` обновлена:

| Поле | Было | Стало |
|------|------|-------|
| `total_eligible_voters` | Врачи со статусом ACTIVE | Врачи со статусом ACTIVE **и** действующей подпиской |
